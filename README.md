# eClass-Sala-Virtual-Aulas-Remotas
 
As salas de aula virtuais são ambientes digitais em que os alunos podem adquirir conhecimento, acessando-os por meio da internet. É por isso que elas também são conhecidas como Ambiente Virtual de Aprendizagem (AVA).

A ideia é simular os aspectos mais importantes de uma classe tradicional e aliá-los às funcionalidades tecnológicas para promover uma melhor aprendizagem, colaboração e integração entre os participantes.

# Sobre o projeto
A tela inicial do sistema recebe os dados necessários para Login inseridos pelo usuário. O sistema reconhece o nível de usuário (Administrador/Professor, aluno) e redireciona o mesmo ao painel específico.<br>
![Capturar_2021_03_21_19_44_26_649](https://user-images.githubusercontent.com/31749933/112041826-9680b400-8b25-11eb-8cb2-d2e1bbbcb812.png)

No painel de administrador, é possível inserir usuários, definir seu nível de acesso, bem como criar salas e inserir membros nas mesmas. É através deste painel ainda que o professor pode inserir sua aula online e interagir com todos os membros.<br>
![Capturar_2021_03_21_19_44_37_423](https://user-images.githubusercontent.com/31749933/112041856-9f718580-8b25-11eb-9824-2ee33cb6ee11.png)

É possível visualizar as aulas em andamento através de um link presente no painel. Ao ser clicado neste, surge um modal contendo a aula atualmente em execução.<br>
![Capturar_2021_03_21_19_45_25_269](https://user-images.githubusercontent.com/31749933/112041862-a0a2b280-8b25-11eb-9999-badcf8056257.png)

Para trocar de aula, o professor deve clicar no botão Troca de Aula. Após aberto o modal com os dados da sala virtual, o mesmo deve inserir o ID do vídeo contendo sua aula. Este ID deve, obrigatoriamente, ser de um vídeo do Youtube. Podem ser inseridos vídeos gravados ou links ao vivo.<br>
![Capturar_2021_03_21_19_45_41_711](https://user-images.githubusercontent.com/31749933/112041866-a1d3df80-8b25-11eb-9cc9-389344d8a9b3.png)

No painel de aluno, o mesmo pode verificar todas as salas virtuais presentes no sistema, contudo, ele somente pode entrar nas quais ele está vinculado. <br>
![Capturar_2021_03_21_19_47_09_374](https://user-images.githubusercontent.com/31749933/112041880-a4ced000-8b25-11eb-91e6-d0504a96bc5d.png)

Ao entrar na sala, o estudante tem acesso ao vídeo atual que está sendo reproduzido, colocado anteriormente pelo seu professor. Através do chat abaixo, o mesmo pode interagir com os professores e alunos daquela sala.<br>
![Capturar_2021_03_21_19_48_23_533](https://user-images.githubusercontent.com/31749933/112041887-a6989380-8b25-11eb-96eb-98acaf2c9884.png)
